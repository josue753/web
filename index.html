Explicación de los estilos utilizados

  • Fondo degradado: Se usa un gradiente de colores vibrantes para el fondo, creando un ambiente llamativo y moderno.
  • Tipografía: Se emplea la fuente Arial para mantener un diseño limpio y legible.
  • Contenedor centrado: El CV está dentro de un contenedor con borde redondeado y sombra para hacerlo más atractivo.
  • Títulos con estilo: Los encabezados (h1, h2) se resaltan con un tamaño grande y colores llamativos para separar las secciones del CV.
  • Información de contacto: Se presenta en un contenedor con diseño en flexbox, asegurando una distribución proporcional de la información.
  • Listas ordenadas: Se utiliza una lista de tipo cuadrado con márgenes para mantener la estructura organizada.
  • Pied de página discreto: Un pie de página sencillo para créditos o información adicional.
